Prepare GitHub secrets

1. Create distribution certificate manually

a. Run Keychain Access

b. Select 'login' from left side

c. Menu 'Certificate Assistant -> Request a Certificate From a Certificate Authority'

d. Input email, select 'Saved to disk'

e. With that file, create a distribution certificate from Apple

f. Download the distribution certificate

g. Create a distribution profile, and download that

2. Double click the downloaded 'distribution.cer', step 1.f

3. In Keychain Access, find the distribution certificate, "My Certificates" tab

4. Export that as '.p12' file. While exporting, enter password. This is P12_PASSWORD secret in GitHub

5. Move to '.p12' file and run: base64 -i that_file.p12 | pbcopy

6. Add that value to GitHub secret as BUILD_CERTIFICATE_BASE64

7. Move to distribution profile file and run: base64 -i provision_file.mobileprovision | pbcopy

8. Add that value to GitHub secret as BUILD_PROVISION_PROFILE_BASE64
